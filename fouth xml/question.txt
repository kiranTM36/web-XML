Create an XML and XSD with the following requirements:

Root element: <university>
Child elements: <student> (can appear multiple times)
Each <student> contains:
rollNumber (attribute, required, integer, 1000–9999)
<name> → string, required
<email> → string, must contain @
<courses> → optional, contains one or more <course> elements
<course> has an attribute code (string, required, pattern: 3 letters + 3 digits, e.g., CSC101)
<course> text is the course name
Maximum 6 courses per student
<university> can contain any number of students.
